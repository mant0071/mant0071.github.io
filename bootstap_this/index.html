Let's Get SASS'y

Preprocessors are important in keeping style sheets managable. Using SASS let's you quickly use features like variables, nesting, inheritance, and other nifty goodies. Bootstrap's source SASS files allow you to pick what you need and map it how you'd like.

We've got OPTIONS

Bootstrap has built in variables that allow you to easily customize and toggle your CSS preferences. Anything from gradients, transitions, spacing, and more can be customized using Bootstrap's variables file.

Living in COLOUR

Bootstrap uses a subset of all colours to create a smaller colour pallette for generating colour schemes (like in the buttons listed before). All of the colours are listed in the SASS map and fully customizable.

Cards

About

Bootstrap's cards are flexible content containers. They can display a wide variety of content, background colours, headers, footers, images, you name it.

Content Types

The sentence you're reading right now is part of the card-body, the building block of the card that gives nice padding. Card titles, links, and text can also be used.
