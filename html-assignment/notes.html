A few HTML things I've learned so far

h1
The h1 is a header tag that we can use for most important heading of the page.
Nav
Nav is used for he navigation of the page, this will contain the links to other pages of the website (as seen below).
Ul
An unordered list is used when the items inside the list do not have a specific order or the order is not important.
Img
Img is the HTML element to display an image on the webpage, which is a singular tag.
A
For creating a link on our webpage we use the anchor tag
